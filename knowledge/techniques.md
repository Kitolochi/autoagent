# Video Engagement Techniques (Top 20)

Compressed from 60-technique catalog. Sorted by versatility (avg value across video types).

## Hooks

**1. Curiosity Gap** | Hype, Explainer, Demo
Open a loop the viewer must close. "You've never seen an app do this." 3-5s, large text, hold without answering. Pair with demo payoff 5-10s later.

**2. Problem-Based** | Tutorial, Compare, Explainer
Show the PROBLEM before product appears. "Struggling with X?" Spend 20-30% of runtime on pain. Product enters as relief at 25-35% mark.

**3. Time-Saving** | Demo, Tutorial
Lead with specific saved time. "Build in 5 min what took 2 hours." Concrete number required.

## Product Techniques

**4. 3D Tilt Entry** | Demo(5), Hype(5)
Product screenshot at rotateY(12deg) + rotateX(3deg) with perspective. Add foreground 3D orb (gold/purple/cyan) with motion trail. Ground shadow beneath orb. The signature product entrance.

**5. UI Reveal** | Demo(5), Hype(5)
Layered panel build creating progressive disclosure. Panels enter staggered (200ms delay), each revealing deeper UI. Builds complexity without overwhelming.

**6. Live Typing** | Demo(5), Tutorial(5)
Character-by-character text input at 40-60 chars/sec. Proves the product works in real-time. Essential for AI/chat tools.

**7. Feature Highlight** | Demo(5), Tutorial(5)
Callout annotations directing attention to specific UI elements. Circle, arrow, or glow pulse on the target. 2-3s per callout.

**8. Cascading Windows** | Hype(5), Recap(5)
Multiple product screenshots stacked with 3D offset. Each enters with slide+scale. Communicates "look at everything we built."

**9. Split View** | Compare(5)
Side-by-side before/after. Left=old, Right=new. Morph or color-wipe between states. Purpose-built for comparison videos.

## Transitions

**10. Beat-Synced Cut** | Hype(5), Brand(5), Recap(5)
Cut on every musical beat. 1 beat = 1 scene at peak energy. Drives rhythm. The single most important production technique for hype videos.

**11. Morph Transition** | Demo(5), Compare(5)
Element from scene A morphs into element in scene B. "Click this button, get this view." Highest avg value (4.0) across all video types.

**12. Geometric Wipe** | Universal(3.5 avg)
Shape-based scene change (circle expand, diagonal slash, grid reveal). 300-500ms. Safe default transition.

**13. Zoom Punch** | Hype(5)
Fast scale(1.0 to 1.3) + blur + cut. 200ms. Impact punctuation between scenes. Max 4 per video.

## Typography & Data

**14. Kinetic Type** | Hype(5), Brand(5)
Words animate individually: scale, rotation, position. Bold 80-120px. 1-5 words per frame. The text IS the visual.

**15. Data Bloom** | Hype(5), Testimonial(5)
Large number animates from 0 to target with easeOutQuart. "$0 to $10M" or "3x faster." Numbers that prove value. Counter + unit + context line.

## Composition & Atmosphere

**16. Light/Dark Alternation** | Pacing strategy, all videos 4+ scenes
Dark = mood/brand, Light = clarity/features. Open dark, close dark. 25-35% light scenes. Prevents visual fatigue. Never 3+ consecutive same-mode scenes.

**17. Negative Space** | Demo, Explainer
60-70% empty frame (solid color, no gradient). ONE focal element off-center. Forces eye to content. Clarity over decoration.

## Narrative Structures

**18. Full-Circle Visual Payoff** | Demo, Compare
Show same UI format twice: first with BAD data (rank absent, score 42/100), then GOOD data (rank #1, score 92/100). Place payoff at 60-80% mark. Identical frame + opposite data = instant proof.

**19. Fear-Validate-Reframe** | Products vs real threats
FEAR (2-3s): state threat. VALIDATE (2-3s): external proof (newspaper, stat). REFRAME (3-5s): product inverts threat. Three steps prevent both fear-mongering and empty promises.

**20. Manifesto-Demo-Manifesto (A-B-A)** | AI launches
Open: 2-3 philosophical text cards on dark bg. Middle: full product demo. Close: 2-3 text cards on LIGHT bg + CTA. The light/dark mode switch signals "meaning section." Optimal for products that are both philosophical and practical.

## Shot Architecture Quick Picks

| Content Goal | Shot Type |
|---|---|
| Single feature | Framed Card (visible border + active edges) |
| Platform overview | Panoramic Blur (full screenshot bg + glass overlay center) |
| Multiple features | Horizontal Scroll Gallery or 3D Text Cloud |
| Cold open / hook | 3D World Reveal (fly-through with floating geometry) |
| Narrative bridge | Kinetic Typography + 3D Companion (orb/cube) |
| Product reveal | Environmental Container (device frame) or Product Tilt + 3D Accent |
| Social proof | Decorative Symmetry (centered content + mirrored flanking elements) |

## Value Rules

1. Identify video type first (Demo/Hype/Tutorial/Compare/Testimonial/Brand/Explainer/Recap)
2. Start with value-5 techniques for that type
3. Fill with value-4 for variety
4. Tutorials: avoid particles, glitch cuts, vortex. Clarity over spectacle
5. Hype: maximize beat-synced cuts, zoom punches, kinetic type
6. Brand: atmospheric techniques (blur, parallax, ambient particles, aurora bg)
7. One visual-proof element per claim. "Looks cool" without message = cut it
